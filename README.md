# 🏀 NBA 生涯模拟器

> 像「人生重开模拟器」一样，模拟一名 NBA 球员从新秀到退役的完整职业生涯。

**基于真实 NBA 数据 · Web 界面 · 每周随机事件 · 玩家主动抉择**

---

## ✨ 功能特性

- 🏀 **530+ 真实 NBA 球员数据**（via nba_api）
- 📅 **赛季模拟**：30 周常规赛，每周 2-4 场比赛
- 🎲 **70+ 随机事件**：伤病、爆发、更衣室冲突、媒体风波…
- ⚡ **主动抉择系统**：每周弹出选择题，不同选择影响生涯走向
- 📊 **真实比赛看板**：完整双方 Box Score（双队真实球员）
- 🧬 **自定义球员**：创建任意球员，指定场均数据（神模式）
- 🏆 **荣誉与历史地位**：MVP、得分王、总冠军、HOF 积分
- 🌐 **Web 界面**：浏览器中运行，深色主题

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据（首次运行，需联网，约 5-10 分钟）

```bash
# 快速模式（仅基础数据，2分钟）
python -m seeding.seed_runner --quick

# 完整模式（含球员详情和球队归属，10分钟）
python -m seeding.seed_runner
```

### 3. 启动游戏

```bash
python run_web.py
```

浏览器自动打开 `http://127.0.0.1:5000`

## 🎮 游戏玩法

| 操作 | 说明 |
|---|---|
| **选择/新建球员** | 从 530+ 真实球员中选，或自建虚构球员 |
| **推进一周** | 模拟 2-4 场比赛，生成随机事件 |
| **主动抉择** | 弹窗选择训练方式、应对风波、合同方向等 |
| **神模式** | 指定场均数据（如 10/10/10/10/10），观察影响 |
| **查看数据** | 逐场数据、比赛 Box Score、历史地位 |

**Space** 键 = 推进一周

## 📁 项目结构

```
nba_sim/
├── run_web.py          # Web 版启动入口
├── web/                # Flask 后端 + HTML/CSS/JS
├── simulation/         # 模拟引擎（赛季、比赛、影响力）
├── events/             # 事件系统（70+ 事件 + 抉择）
├── database/           # SQLite + Repository 层
└── seeding/            # NBA 数据导入
```
